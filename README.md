# Quantum Portfolio Optimization World Record
The World Record Breaking Attempt At Portfolio Optimization Using Automatski's Quantum Annealers

All attempts peak at around 50 Stock Portfolios. Q-CHOP has done 250 Stock Portfolios but it is a Hybrid Algorithm. This is a pure quantum algorithm without any classical processing.

## About

This repository contains an implementation of Portfolio Optimization using Quantum Annealers created by [Automatski](https://automatski.com). Automatski's Quantum Annealers support upto 500 million spin up-down Boolean Variables.

## Installation

```sh
pip install -r requirements.txt
```

## Execution

On Windows run the batch file. On Linux/Mac see below

```sh
python app.py
```
